## Usage

### AI Agents
Discover and use skills automatically. Each `SKILL.md` contains:
- YAML frontmatter with purpose
- Language-specific guidelines
- Best practices and examples

### Developers
1. Start with [AGENTS.md](AGENTS.md) for available skills
2. Navigate to skill folders for detailed guidelines
3. Follow standards when writing code/docs
4. Ensure consistency with project linters/formatters

### Projects
- Reference skills in project documentation
- Customize/extend for project-specific needs
- Keep skills updated as standards evolve
